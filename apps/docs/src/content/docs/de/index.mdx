## Warum SpriteBox?

- **100% Kostenlos & Open Source** - Keine Werbung, keine Paywalls, kein Tracking
- **Kein Account nötig** - Einfach Link öffnen und spielen
- **Funktioniert überall** - Desktop, Handy, Tablet
- **Mehrsprachig** - Englisch und Deutsch
- **Ein Befehl zum Hosten** - Eigenen Server in Sekunden starten
